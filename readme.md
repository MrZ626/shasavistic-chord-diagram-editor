## Installation

### Windows

1. Clone/Download this repository
2. Download love2d engine from https://love2d.org/
3. Drag the project folder onto `love.exe`

### Linux

1. Clone/Download this repository
2. Get love2d engine from your package manager
3. Run `love .` in the project folder
